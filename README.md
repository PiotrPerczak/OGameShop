## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/PiotrPerczak/OGameShop.git
   cd OGameShop
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run the app:**
   ```bash
   npm run dev
   ```
4. **Open in your browser:**
   [http://localhost:3000](http://localhost:3000)
